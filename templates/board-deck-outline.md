# Board Deck Outline

## Recommended Structure (45-60 min meeting)

### 1. Headline Metrics (2 min)
- MRR / ARR
- Growth rate (MoM, YoY)
- Runway remaining
- Key ratio: LTV/CAC

### 2. Executive Summary (3 min)
- What happened since last board meeting (3 bullets max)
- Biggest win
- Biggest concern
- Strategic question for the board

### 3. Financial Performance (10 min)

| Metric | Actual | Budget | Variance | Commentary |
|--------|--------|--------|----------|------------|
| Revenue | | | | |
| Gross Margin | | | | |
| Operating Expenses | | | | |
| Net Burn | | | | |
| Cash Position | | | | |
| Runway | | | | |

### 4. Product & Engineering (10 min)
- Key features shipped
- Roadmap progress vs. plan
- Technical debt / infrastructure update
- Key product metrics (activation, retention, NPS)

### 5. Sales & Marketing (10 min)
- Pipeline overview
- Win/loss analysis
- Customer acquisition metrics
- Marketing performance by channel

### 6. Team & Organization (5 min)
- Headcount (plan vs. actual)
- Key hires made / needed
- Organizational health
- Culture / engagement

### 7. Strategic Discussion (15 min)
<!-- One or two strategic topics for board input -->
- Topic 1: [Description + options + recommendation]
- Topic 2: [Description + options + recommendation]

### 8. Asks from the Board (5 min)
- Specific introductions needed
- Strategic guidance on X
- Approval of Y

### Appendix (for reference, not presentation)
- Detailed financial statements
- Product roadmap detail
- Competitive landscape update
- Cap table
