# SEO Reference

## Technical SEO Checklist
- XML sitemap submitted to Google Search Console
- robots.txt properly configured
- HTTPS everywhere
- Core Web Vitals passing (LCP <2.5s, FID <100ms, CLS <0.1)
- Mobile-friendly design
- Clean URL structure
- Canonical tags on all pages
- No broken links (404s)
- Internal linking structure (max 3 clicks to any page)
- Schema markup (Organization, Product, FAQ, Article, Breadcrumb)

## On-Page SEO Checklist (Per Page)
- [ ] Unique title tag (50-60 characters, primary keyword)
- [ ] Compelling meta description (150-160 characters)
- [ ] One H1 with primary keyword
- [ ] Logical heading hierarchy (H1→H2→H3)
- [ ] Primary keyword in first 100 words
- [ ] Alt text on all images
- [ ] Internal links to related content (2-5 per post)
- [ ] External links to authoritative sources
- [ ] Content matches search intent
- [ ] Content is comprehensive (covers the topic thoroughly)

## Keyword Research Framework

### Priority Matrix
| | High Volume | Low Volume |
|---|------------|-----------|
| **High Intent** | Priority 1 | Priority 2 |
| **Low Intent** | Priority 3 | Skip |

### Bottom-of-Funnel First
Start with keywords closest to conversion:
1. "[Product] pricing" / "[Product] vs [Competitor]"
2. "Best [category] for [use case]"
3. "How to [solve problem your product solves]"
4. "[Category] tools/software"
5. Informational content (broader topics)

## Link Building Strategies
1. **Create linkable assets:** Tools, calculators, original research, templates
2. **Guest posting:** Write for industry blogs with relevant audience
3. **HARO/journalist queries:** Provide expert quotes for articles
4. **Broken link building:** Find broken links on relevant sites, offer your content
5. **Partner page links:** Get listed on integration/partner directories
6. **Resource page links:** Find resource pages in your niche, pitch inclusion
