# Investor Reporting

## Monthly Update Best Practices

### Format
Keep it short — investors get dozens of updates. 500 words max.

### Structure
1. **TL;DR** (2 sentences): Biggest win + biggest challenge
2. **Key metrics** (table): MRR, growth, churn, runway, one product metric
3. **Wins** (3-5 bullets): What went well
4. **Challenges** (2-3 bullets): Be honest — investors respect transparency
5. **Asks** (1-3 bullets): Specific ways they can help (intros, advice)
6. **Looking ahead** (2-3 bullets): Top priorities for next month

### Rules
- Send on the same date every month (consistency builds trust)
- Bad news travels fast — never surprise investors in a board meeting
- Be specific with asks ("intro to VP Sales at Stripe" not "sales intros")
- Include a link to a live metrics dashboard if possible
- Reply to any investor questions within 24 hours

## Data Room Setup

### Standard Contents

| Category | Documents |
|----------|-----------|
| **Company** | Pitch deck, executive summary, company overview |
| **Financial** | Financial model, cap table, tax returns, bank statements |
| **Legal** | Articles of incorporation, operating agreement, IP assignments |
| **Product** | Product demo/screenshots, technical architecture, roadmap |
| **Team** | Founder bios, org chart, key hire profiles |
| **Market** | Market research, competitive analysis, customer references |
| **Metrics** | KPI dashboard, cohort analysis, funnel metrics |

### Data Room Tools
- DocSend (tracking who views what)
- Google Drive (simple, free)
- Notion (organized, shareable)

### Security
- Use view-only links (no downloads for sensitive docs)
- Track who opens what and when
- Watermark documents if needed
- Revoke access after round closes

## Board Financial Package

### What Boards Want to See

1. **Dashboard page:** 8-10 key metrics with trends
2. **P&L:** Actual vs budget, with variance commentary
3. **Cash flow:** Cash position, burn rate, runway
4. **ARR bridge:** How ARR changed (new, expansion, churn)
5. **Unit economics:** CAC, LTV, payback by channel
6. **Forecast:** Updated projections with key assumptions
7. **Headcount:** Plan vs actual, upcoming hires

### Presentation Tips
- Lead with the story, not the spreadsheet
- Highlight variances and explain them
- Show trends (3-6 month view), not just snapshots
- Have detailed backup ready but don't present it
- End with the decision or input you need from the board
