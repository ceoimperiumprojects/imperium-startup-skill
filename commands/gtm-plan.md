# Go-to-Market Plan Generator

You are building a GTM plan using the Imperium Startup framework.

## Information Gathering

Ask the user for:
1. **Product**: What are you launching?
2. **Stage**: New product launch / New market entry / Feature launch / Pivot
3. **Target market**: Who is the ideal customer?
4. **Current state**: Any existing customers, revenue, or traction?
5. **Budget**: What resources are available? (money, team, time)
6. **Timeline**: When does this need to launch?

## GTM Plan Creation

Reference template: `templates/gtm-plan.md`

### 1. ICP Definition
Build a detailed Ideal Customer Profile:
- Firmographics (industry, size, revenue, geography)
- Technographics (what tools they use)
- Behavioral signals (what they're searching for, pain indicators)
- Buyer persona (who makes the decision, who influences)

### 2. Positioning & Messaging
Using Geoffrey Moore's framework:
- **For** [target customer]
- **Who** [need/pain point]
- **Our product is a** [category]
- **That** [key benefit]
- **Unlike** [alternatives]
- **We** [key differentiator]

### 3. Channel Strategy
Evaluate and prioritize channels:

| Channel | Effort | Cost | Time to Results | Fit |
|---------|--------|------|----------------|-----|
| Content/SEO | High | Low | 3-6 months | |
| Paid Ads | Low | High | Immediate | |
| Cold Outbound | Medium | Low | 1-3 months | |
| Partnerships | Medium | Low | 2-4 months | |
| Community | High | Low | 3-6 months | |
| Events | Medium | High | 1-2 months | |
| Product-led | High | Low | 1-3 months | |

Pick 2-3 primary channels for Phase 1.

### 4. Sales Motion
Determine the right sales approach:
- **Self-serve**: Low ACV, high volume, product-led
- **Sales-assisted**: Medium ACV, product + human help
- **Enterprise**: High ACV, dedicated sales cycle
- **Hybrid**: Self-serve for small, sales for large

### 5. Launch Plan
Create a phased timeline:
- **T-30**: Pre-launch (build anticipation, content, partnerships)
- **T-0**: Launch (coordinated push across all channels)
- **T+30**: Post-launch (optimize, iterate, double down on what works)

### 6. Metrics Framework
Define what success looks like:

| Phase | Key Metric | Target |
|-------|-----------|--------|
| Awareness | Website traffic, impressions | |
| Acquisition | Signups, trials started | |
| Activation | First value moment reached | |
| Revenue | Paying customers, MRR | |
| Retention | Churn rate, NRR | |

## Output Format

Deliver a complete GTM plan document covering:
1. Executive summary (1 paragraph)
2. ICP and buyer persona
3. Positioning statement and key messages
4. Channel strategy with prioritization
5. Sales motion design
6. Phased launch timeline
7. Budget allocation
8. Success metrics and targets
9. Risk factors and contingencies
