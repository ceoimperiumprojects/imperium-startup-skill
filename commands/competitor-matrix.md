# Competitive Analysis Workflow

You are building a competitive analysis using the Imperium Startup framework.

## Step 1: Gather Context

Ask the user:
1. What is your product/service?
2. Who is your target customer?
3. Who do you consider your main competitors? (or ask to research)
4. What do you think your key differentiators are?

## Step 2: Competitor Identification

If the user hasn't listed competitors, research and identify:
- **Direct competitors**: Same solution to same problem
- **Indirect competitors**: Different solution to same problem
- **Potential competitors**: Companies that could enter this space
- **Substitute competitors**: What people do instead (including "nothing")

For each, gather:
- Company overview (founded, funding, team size)
- Product description and key features
- Pricing model and tiers
- Target customer segment
- Strengths and weaknesses
- Recent news/developments

## Step 3: Feature Comparison Matrix

Create a detailed feature-by-feature comparison.
Reference template: `templates/competitive-matrix.md`

Categorize features as:
- **Table stakes**: Must-have to compete (everyone has these)
- **Differentiators**: Where products diverge
- **Unique**: Only one product has this

## Step 4: Positioning Map

Create a 2x2 positioning map.

Help the user choose axes that:
- Represent dimensions customers actually care about
- Put their product in the top-right quadrant
- Are defensible (not arbitrary)

Suggested axis pairs:
- Ease of use ↔ Power/functionality
- Price ↔ Value delivered
- Self-serve ↔ Enterprise
- Speed to value ↔ Depth of solution

## Step 5: Battlecards

For each major competitor, create a sales battlecard:

### When prospect mentions [Competitor]:
- **Acknowledge**: What they're good at (builds credibility)
- **Differentiate**: Where you're different/better
- **Prove**: Customer quotes, data points, case studies
- **Landmine**: Questions that expose competitor weaknesses
- **Objection handling**: Common objections and responses

## Step 6: Strategic Insights

Analyze the landscape for:
- **White space**: Unserved segments or unmet needs
- **Trends**: Where the market is heading
- **Threats**: Competitive moves to watch
- **Opportunities**: Strategic advantages to exploit

## Output Format

Deliver:
1. Competitor overview table
2. Feature comparison matrix
3. Positioning map (2x2 with placement rationale)
4. Battlecard for each major competitor
5. Strategic insights and recommendations
6. Recommended monitoring plan (what to track, how often)
