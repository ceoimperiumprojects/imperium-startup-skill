# SEO Checklist for Startups

## Technical SEO Foundation

### Site Structure
- [ ] XML sitemap generated and submitted to Google Search Console
- [ ] robots.txt properly configured
- [ ] Clean URL structure (no query params for key pages)
- [ ] HTTPS everywhere (no mixed content)
- [ ] Mobile-friendly / responsive design
- [ ] Page speed: Core Web Vitals passing (LCP < 2.5s, FID < 100ms, CLS < 0.1)
- [ ] No broken links (404s)
- [ ] Canonical tags on all pages
- [ ] Hreflang tags (if multi-language)

### Crawlability
- [ ] Google Search Console verified
- [ ] No important pages blocked by robots.txt
- [ ] Internal linking structure logical (max 3 clicks to any page)
- [ ] Breadcrumb navigation implemented
- [ ] Pagination handled correctly (rel=next/prev or infinite scroll with crawlable links)

### Schema Markup
- [ ] Organization schema on homepage
- [ ] Product/Service schema on relevant pages
- [ ] FAQ schema on FAQ pages
- [ ] Article schema on blog posts
- [ ] Breadcrumb schema
- [ ] Review/Rating schema (if applicable)

## On-Page SEO

### Every Page Should Have
- [ ] Unique, descriptive title tag (50-60 characters)
- [ ] Compelling meta description (150-160 characters)
- [ ] One H1 tag per page (includes primary keyword)
- [ ] Logical heading hierarchy (H1 → H2 → H3)
- [ ] Alt text on all images
- [ ] Internal links to related content
- [ ] Outbound links to authoritative sources (where relevant)

### Content Quality
- [ ] Primary keyword in title, H1, first 100 words, and URL
- [ ] Content matches search intent (informational vs. transactional)
- [ ] Comprehensive coverage of topic (no thin content)
- [ ] Unique content (no duplicate/copied content)
- [ ] Updated regularly (especially competitive keywords)

## Content Strategy

### Keyword Research
- [ ] Seed keyword list built from customer interviews
- [ ] Competitor keyword gap analysis done
- [ ] Keywords prioritized by: volume × intent × difficulty
- [ ] Content calendar mapped to keyword clusters
- [ ] Bottom-of-funnel keywords prioritized first (closer to conversion)

### Content Types to Build
- [ ] Product/feature pages (transactional intent)
- [ ] Comparison pages ("X vs Y", "X alternatives")
- [ ] "How to" guides (informational, top of funnel)
- [ ] Case studies (social proof + long-tail keywords)
- [ ] Glossary/definition pages (high-volume informational)
- [ ] Templates/tools pages (link magnets)

### Content Production
- [ ] Publishing cadence: minimum 2-4 posts/month
- [ ] Each piece targets a specific keyword cluster
- [ ] Internal linking strategy between content pieces
- [ ] Content refresh schedule for aging posts

## Off-Page SEO

### Link Building
- [ ] Guest posting on industry blogs
- [ ] HARO / journalist query responses
- [ ] Create linkable assets (tools, research, templates)
- [ ] Partner/integration page backlinks
- [ ] Industry directory listings
- [ ] Broken link building campaigns

### Local SEO (if applicable)
- [ ] Google Business Profile claimed and optimized
- [ ] NAP (Name, Address, Phone) consistent across web
- [ ] Local directory listings
- [ ] Customer reviews strategy

## Measurement

### Key Metrics
- [ ] Organic traffic (Google Analytics)
- [ ] Keyword rankings (Ahrefs, SEMrush, or similar)
- [ ] Organic conversions (signups, demos, purchases)
- [ ] Page-level performance (which pages drive results)
- [ ] Backlink growth (new referring domains/month)
- [ ] Core Web Vitals scores

### Monthly Review
- [ ] Traffic trend vs. previous month
- [ ] Top performing content (traffic + conversions)
- [ ] New keyword rankings gained
- [ ] Technical issues flagged in Search Console
- [ ] Content gaps identified
- [ ] Competitor movement tracked
