# Product Launch Checklist

## Pre-Launch (T-30 to T-7)

### Product
- [ ] Core features complete and tested
- [ ] Bug sweep done — no P0/P1 bugs open
- [ ] Performance tested under expected load
- [ ] Security review completed
- [ ] Analytics/tracking implemented (key events, funnels)
- [ ] Onboarding flow tested with real users
- [ ] Error handling and edge cases covered
- [ ] Mobile/responsive tested (if applicable)

### Infrastructure
- [ ] Production environment ready
- [ ] Monitoring and alerting configured
- [ ] Backup and disaster recovery plan in place
- [ ] SSL/security certificates valid
- [ ] CDN configured (if applicable)
- [ ] Scaling plan for launch traffic

### Legal & Compliance
- [ ] Terms of Service published
- [ ] Privacy Policy published
- [ ] Cookie consent (GDPR if applicable)
- [ ] Data processing agreements signed
- [ ] Any required certifications obtained

### Marketing
- [ ] Landing page live and tested
- [ ] Email sequences ready (welcome, onboarding, activation)
- [ ] Social media content calendar prepared
- [ ] Blog post / announcement draft ready
- [ ] Press kit prepared (logo, screenshots, founder bios)
- [ ] Product Hunt page drafted (if applicable)
- [ ] Influencer/partner outreach started

### Sales
- [ ] Pricing page live
- [ ] Payment processing tested (test transactions)
- [ ] Demo script prepared
- [ ] FAQ / knowledge base published
- [ ] Battlecards vs. competitors ready

### Support
- [ ] Support channels set up (email, chat, docs)
- [ ] Help documentation written
- [ ] Support team briefed on product
- [ ] Escalation process defined
- [ ] Response time SLAs set

## Launch Day (T-0)

### Go-Live
- [ ] Deploy to production
- [ ] Verify all critical flows work
- [ ] Monitor error rates and performance
- [ ] Team on standby for issues

### Announce
- [ ] Send announcement email
- [ ] Publish blog post
- [ ] Post on social media (Twitter, LinkedIn, etc.)
- [ ] Submit to Product Hunt (if applicable)
- [ ] Notify press/media contacts
- [ ] Alert partners and affiliates
- [ ] Post in relevant communities (Hacker News, Reddit, etc.)

### Monitor
- [ ] Watch analytics dashboard
- [ ] Monitor social media mentions
- [ ] Track signup/conversion funnel
- [ ] Respond to early feedback quickly
- [ ] Document issues for post-launch fixes

## Post-Launch (T+1 to T+30)

### Week 1
- [ ] Review Day 1 metrics vs. targets
- [ ] Fix any critical bugs discovered
- [ ] Gather and organize user feedback
- [ ] Send follow-up to press/media
- [ ] Publish "thank you" / launch recap

### Week 2-4
- [ ] Analyze activation and retention metrics
- [ ] Identify drop-off points in funnel
- [ ] Ship quick wins based on feedback
- [ ] Develop first case study
- [ ] Start SEO content engine
- [ ] Test paid acquisition channels
- [ ] Internal retrospective on launch process
