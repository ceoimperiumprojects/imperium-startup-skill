# Fundraising Checklist

## Stage 1: Preparation (4-8 weeks before)

### Story & Materials
- [ ] Pitch deck ready (10-12 slides)
- [ ] Executive summary (1-pager)
- [ ] Financial model (3-year projections)
- [ ] Cap table current and clean
- [ ] Demo ready (working product or prototype)
- [ ] Data room set up (Google Drive or DocSend)

### Data Room Contents
- [ ] Pitch deck (PDF)
- [ ] Financial model (spreadsheet)
- [ ] Cap table
- [ ] Articles of incorporation
- [ ] Team bios
- [ ] Product demo / screenshots
- [ ] Key metrics dashboard
- [ ] Customer references (if available)
- [ ] Technical architecture overview
- [ ] Market research / competitive analysis

### Know Your Numbers
- [ ] MRR / ARR (or pre-revenue narrative)
- [ ] Growth rate (MoM)
- [ ] CAC and LTV (or estimates)
- [ ] Burn rate and runway
- [ ] Total addressable market (bottom-up)
- [ ] Amount raising and use of funds
- [ ] Target valuation and dilution

### Practice
- [ ] Pitch rehearsed 10+ times
- [ ] Timed to under 15 minutes
- [ ] Q&A prep: top 50 investor questions answered
- [ ] "Why now?" answer crisp
- [ ] "Why you?" answer compelling
- [ ] Rejection responses prepared

## Stage 2: Targeting (2-4 weeks before)

### Investor Research
- [ ] List of 50-100 target investors
- [ ] Filtered by: stage, sector, check size, geography
- [ ] Research each: portfolio, thesis, recent investments
- [ ] Identify warm intro paths for top 30
- [ ] Tier investors: A (dream), B (good fit), C (backup)

### Warm Intros
- [ ] Request intros from existing network
- [ ] Reach out to portfolio founders for references
- [ ] Leverage advisors for introductions
- [ ] Use LinkedIn/Twitter for relationship building
- [ ] Attend relevant events for face-to-face

## Stage 3: Active Fundraise (4-8 weeks)

### Process Management
- [ ] CRM / spreadsheet tracking all conversations
- [ ] Track: investor, stage, next step, timeline, notes
- [ ] Start with B-tier investors (practice rounds)
- [ ] Create urgency: parallel processes, momentum signals
- [ ] Weekly pipeline review

### Meeting Cadence
- [ ] First meetings: 15-20 per week
- [ ] Partner meetings: as they come
- [ ] Follow-ups: within 24 hours
- [ ] Updates: weekly to interested investors

### Signals to Watch
- [ ] "Let me introduce you to my partner" = good
- [ ] "Let's stay in touch" = soft no
- [ ] "We'd like to lead" = great, move fast
- [ ] Multiple term sheets = best position

## Stage 4: Closing (2-4 weeks)

### Term Sheet
- [ ] Term sheet received
- [ ] Lawyer reviewed terms
- [ ] Key terms negotiated (valuation, board seats, pro-rata, liquidation preference)
- [ ] Understand: standard vs. non-standard terms
- [ ] Compare multiple term sheets (if available)

### Due Diligence
- [ ] Data room shared with lead investor
- [ ] Reference checks (they call your customers/former colleagues)
- [ ] Technical due diligence (if applicable)
- [ ] Legal due diligence
- [ ] Financial audit (if applicable)

### Legal & Close
- [ ] Lawyer engaged for closing docs
- [ ] SAFE / Convertible Note / Priced Round docs prepared
- [ ] Board composition agreed
- [ ] Side letters (if any) reviewed
- [ ] Wire instructions confirmed
- [ ] Announce internally first, then publicly

## Post-Close

- [ ] Send thank you to all investors (even those who passed)
- [ ] Set up monthly investor update cadence
- [ ] Update cap table
- [ ] File any required regulatory paperwork
- [ ] Celebrate with the team
